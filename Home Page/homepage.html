Website Description

This website is an assignment I was given in one of the subjects of my course.
Along with the Home Page which contains an About Me section, my e-Portfolio includes an CV page which highlights some of my IT skills and People skills. The CV page also includes an attached copy of my resume which has description of various part-time jobs I have taken over the years.
In addition to the CV page, my e-portfolio also includes a Favourites Page which lists few of my favourite TV-series and Movies.
